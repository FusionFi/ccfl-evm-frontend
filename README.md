# ccfl-evm-frontend

The official frontend application for CCFL on EVM chains

# require

Node 18.18.2

## Getting Started

# install node_modules

```bash
yarn install (recommend)
#or
npm install
# or
pnpm install

```

# run dev

```bash
yarn dev (recommend)
#or
npm run dev
# or
pnpm dev
```

# build production

```bash
yarn run build (recommend)
#or
npm run build
# or
pnpm build
```
